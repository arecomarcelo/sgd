PASSO 4 — ATUALIZAR service.py
================================================================================

Substituir imports e DB_CONFIG hardcoded pelo padrão abaixo:

--- ANTES ---
  class DataService:
      def __init__(self):
          self.DB_CONFIG = {
              "dbname": "nome_banco",
              "user": "postgres",
              "password": "senha_hardcoded",
              "host": "192.168.1.1",
              "port": "5432",
          }

--- DEPOIS ---
  import os

  import streamlit as st
  from dotenv import load_dotenv

  load_dotenv()


  def _get_db_secret(key: str, default: str = "") -> str:
      """Lê credencial de st.secrets (Streamlit Cloud) ou os.environ (local/.env)."""
      try:
          return st.secrets["database"].get(key, os.environ.get(key, default))
      except Exception:
          return os.environ.get(key, default)


  class DataService:
      def __init__(self):
          self.DB_CONFIG = {
              "dbname": _get_db_secret("DB_NAME", "nome_banco"),
              "user": _get_db_secret("DB_USER", "postgres"),
              "password": _get_db_secret("DB_PASSWORD"),
              "host": _get_db_secret("DB_HOST", "192.168.1.1"),
              "port": _get_db_secret("DB_PORT", "5432"),
          }

LÓGICA DO HELPER _get_db_secret():
  1. Tenta ler de st.secrets["database"] (funciona no Streamlit Cloud e local com secrets.toml)
  2. Se st.secrets não estiver disponível, cai para os.environ (lido do .env via load_dotenv)
  3. Se nenhum dos dois tiver o valor, retorna o default fornecido

PASSO 6 — ATUALIZAR config/settings.py (se existir dataclass de configuração)
================================================================================

--- ANTES ---
  from dataclasses import dataclass

  @dataclass
  class DatabaseConfig:
      host: str = "192.168.1.1"
      port: int = 5432
      database: str = "nome_banco"
      username: str = "postgres"
      password: str = "senha_hardcoded"

--- DEPOIS ---
  import os
  from dataclasses import dataclass, field

  from dotenv import load_dotenv

  load_dotenv()

  @dataclass
  class DatabaseConfig:
      host: str = field(default_factory=lambda: os.environ.get("DB_HOST", "192.168.1.1"))
      port: int = field(default_factory=lambda: int(os.environ.get("DB_PORT", "5432")))
      database: str = field(default_factory=lambda: os.environ.get("DB_NAME", "nome_banco"))
      username: str = field(default_factory=lambda: os.environ.get("DB_USER", "postgres"))
      password: str = field(default_factory=lambda: os.environ.get("DB_PASSWORD", ""))

  NOTA: usar field(default_factory=...) é necessário em dataclasses porque
  os.environ.get() precisa ser avaliado em tempo de execução, não em tempo de
  definição da classe.

PASSO 8 — CONFIGURAR SECRETS NO STREAMLIT CLOUD (produção)

2. Selecionar o app
3. Ir em: Settings → Secrets

  [database]
  DB_NAME = "nome_do_banco"
  DB_USER = "usuario"
  DB_PASSWORD = "sua_senha_real"
  DB_HOST = "ip_ou_host"
  DB_PORT = "5432"

5. O app reinicia automaticamente após salvar.
